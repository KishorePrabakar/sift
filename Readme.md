# Sift – Online Shopping Assistant

Quick tool to compare 2–3 Amazon products.

Paste links → see prices, ratings, features side-by-side → find the best value fast.

### Tech
- React + Node.js/Express

### Quick Start
1. `cd backend && npm install && npm start`
2. `cd frontend && npm install && npm start`

Open http://localhost:3000

Built to help shoppers decide faster.
